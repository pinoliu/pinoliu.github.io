## 简介

本文主要介绍一些 Linux系统 中常用的快捷键，可以有效提升工作效率。



## 实用技巧

当要临时去其它目录时，可输入 `pwd` 保存当前目录的信息，当要返回当前目录时，只需要输入 `cd -` 即可。



## Linux快捷键

| 命令       | 效果                       |
| ---------- | -------------------------- |
| `Ctrl`+`a` | 将光标移动到行首           |
| `Ctrl`+`e` | 将光标移动到行尾           |
| `Ctrl`+`f` | 将光标向右移动一个字符     |
| `Ctrl`+`b` | 将光标向左移动一个字符     |
| `Alt`+`f`  | 将光标向右移动一个单词     |
| `Alt`+`b`  | 将光标向左移动一个单词     |
| `Ctrl`+`w` | 删除光标前面的一个单词     |
| `Ctrl`+`l` | 清理屏幕，只显示当前命令行 |
| `tab`      | 补全命令                   |



## VIM快捷键

| 命令         | 效果                   |
| ------------ | ---------------------- |
| `gg` 或 `G0` | 移动到文章开头         |
| `G`          | 移动到文章结尾         |
| `#G`         | 移动到文章第#行        |
| `$`          | 移动到光标所在的行尾   |
| `^`          | 移动到光标所在的行首   |
| `w`          | 移动到下一个单词的开头 |
| `e`          | 移动到下一个单词的结尾 |
| `b`          | 移动到上一个单词的开头 |
| `dd`         | 删除光标所在行         |
| `#dd`        | 删除从光标开始#行      |
| `u`          | 撤销一次修改           |
| `Ctrl`+` r`  | 反撤销一次修改         |
